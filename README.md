

A new Flutter project.

This is a music player project made in Flutter capable of reading music, importing all your music from your phone, as well as videos, for this project to work, you will need to install all the dependencies of this project with the "flutter pub get" command and you can see the preview, but you must first install a device on your phone, as well as the devices that the flutter SDK and Android studio, activate the "adb device" of your phone then start the project with the command "flutter run" with the device of your phone
